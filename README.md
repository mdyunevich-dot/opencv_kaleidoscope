# opencv_kaleidoscope
# Приложение «Калейдоскоп» на Tkinter и OpenCV (Вариант 25)

## Описание проекта
Программа представляет собой настольное приложение, которое позволяет загружать изображения и применять к ним эффекты зеркальной симметрии, создавая калейдоскопические кадры двух типов:
1. Отражение левой половины матрицы изображения направо.
2. Отражение верхней половины матрицы изображения вниз.

## Архитектура проекта
Проект реализован в соответствии с принципами ООП и состоит из двух ключевых классов:
*   **`Processor`** — отвечает за бизнес-логику приложения: безопасное чтение/сохранение файлов через бинарные потоки, разрезание матриц NumPy, зеркальный переворот методами OpenCV (`cv2.flip`) и конкатенацию массивов (`np.hstack`/`np.vstack`).
*   **`App`** — отвечает за графический интерфейс Tkinter: построение двухвкладочного окна (`ttk.Notebook`), размещение элементов управления (`Radiobutton`), мгновенный пересчет симметрии «на лету» и рендеринг изображений.

## Инструкция по использованию
1. Запустите скрипт `main.py`.
2. На вкладке **«Управление»** нажмите кнопку **«Открыть изображение»** и выберите любой графический файл (`.jpg`, `.png`, `.bmp`).
3. С помощью переключателей `Radiobutton` выберите необходимый режим симметрии (горизонтальный или вертикальный). Пересчет узора происходит мгновенно.
4. Перейдите на вкладку **«Просмотр результата»**, чтобы увидеть готовый калейдоскопический кадр.
5. Нажмите кнопку **«Сохранить результат»** на первой вкладке, чтобы экспортировать полученное изображение.
